
# Tic Tac Toe Game

multiplayer tic tac toe game where i used react , node-js and socketio


## Screenshots

![App Screenshot](https://i.ibb.co/fFCmqTR/tic-tac-toe.png)


## Run Locally

Clone the project

```bash
  git clone https://github.com/Phantom804/expense-tracker.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Start the client

```bash
  npm run server
```
